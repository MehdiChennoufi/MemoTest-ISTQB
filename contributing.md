# Comment contribuer

Bienvenue ! Voici comment vous pouvez aider :

- 🐛 Signaler une erreur → ouvrir une Issue
- ✏️ Corriger du contenu → Pull Request
- 💡 Suggérer une amélioration → ouvrir une Issue

## Règles
- Respecter la licence CC BY-NC-SA 4.0
- Aucune commercialisation
- Les ajouts de contenu doivent s'appuyer sur les syllabus officiels ISTQB
